# DeviceAppType (enum)
DeviceAppType serves destination types for devices.
### Service
+ [AdGroupService](../services/AdGroupService.md)

| Value | Description | 
|---|---|
| APP| Ads for apps. |
| WEB| Ads for the Web. |
| NONE| Non-targeting Posted on all devices. |
