# ComparisionResult(enum)
ComparisionResultは、前日、前週、または前月との比較結果を表します。
 
### Service
+ [StatsService](../services/StatsService.md)

| Enumeration | Type | Description |
|---|---|---|
| DOWN | xsd:string | 下降しました。 |
| UP | xsd:string | 上昇しました。 |
| KEEP | xsd:string | 変化はありません。 |
