# AdProductStatus (enum)
AdProductStatusオブジェクトは、広告の購入可否を表します。
### Service
+ [AccountAdProductService](../services/AccountAdProductService.md)

| 値 | 説明 | 
|---|---|
| OPEN| 購入可能です。 |
| SHUT| 購入不可能です。 |
