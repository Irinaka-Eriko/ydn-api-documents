# TargetListType (enum)
TargetListTypeは、ターゲティングの種別を表します。
### Service
+ [RetargetingListService](../services/RetargetingListService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| DEFAULT_LIST| string| デフォルトリストです。 |
| RULE| string| ルール(条件)です。 |
| COMBINATION| string| 組み合わせです。 |
| SIMILARITY| string| 類似ユーザリストです。 |
