# AdScheduleTarget
AdScheduleTargetオブジェクトは、時間帯ターゲティング設定を表します。
### Service
+ [AdGroupTargetService](../services/AdGroupTargetService.md)

| フィールド | データ型 | 説明 | 
|---|---|---|
| type| enum <a href="../data/TargetType.md">TargetType</a>| ターゲティング設定です。 |
| dayOfWeek| enum <a href="../data/DayOfWeek.md">DayOfWeek</a>| 曜日です。 |
| startHour| xsd:int| 開始時刻です。 |
| endHour| xsd:int| 終了時刻です。 |
