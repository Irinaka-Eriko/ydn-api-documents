# AccountStatus (enum)
AccountStatusは、アカウントの登録状況を表します。
### Service
+ [AccountService](../services/AccountService.md)

| 値 | 説明 | 
|---|---|
| INPROGRESS| 登録待ちです。 |
| WAIT_DECIDE| 確定待ちです。 |
| SERVING| サービス中です。 |
| PENDING| 審査中です。 |
| ENDED| 終了です。 |
