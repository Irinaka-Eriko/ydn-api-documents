# RetargetingTag
RetargetingTagオブジェクトは、サイトリターゲティングのタグ情報を表します。
### Service
+ [RetargetingTagService](../services/RetargetingTagService.md)

| フィールド | データ型 | 説明 | 
|---|---|---|
| retargetingTagId| xsd: string| サイトリターゲティングのタグIDです。 |
| accountId| xsd: long| アカウントIDです。 |
| approvalStatus| enum <a href="../data/RetargetingTagApprovalStatus.md">RetargetingTagApprovalStatus</a>| サイトリターゲティングのステータスです。 |
