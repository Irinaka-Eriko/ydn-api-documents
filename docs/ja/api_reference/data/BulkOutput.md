# BulkOutput (enum)
BulkOutputは、バルク処理の出力フォーマットを表します。
### Service
+ [BulkService](../services/BulkService.md)

| 値 | 説明 | 
|---|---|
| XML| XMLフォーマットです。 |
| CSV| カンマ区分されたフォーマットです。 |
| ZIPPED_XML| ZIP圧縮されたXMLフォーマットです。 |
| ZIPPED_CSV| ZIP圧縮されたカンマ区分されたフォーマットです。 |
