# AdStyle (enum)
AdStyleは掲載フォーマットの種別を表します。
### Service
+ [AdGroupAdService](../services/AdGroupAdService.md)

| 値 | 説明 | 
|---|---|
| TEXT| テキストの掲載フォーマットです。 |
| IMAGE| 画像の掲載フォーマットです。 |
| ANIMATION_IMAGE| アニメーション画像の掲載フォーマットです。 |
