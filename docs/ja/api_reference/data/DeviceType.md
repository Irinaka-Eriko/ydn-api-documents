# DeviceType (enum)
DeviceTypeは、配信先デバイスの種類を表します。
### Service
+ [AdGroupService](../services/AdGroupService.md)

| 値 | 説明 | 
|---|---|
| DESKTOP| PCです。 |
| WAP_MOBILE| 従来型携帯電話です。 |
| SMARTPHONE| スマートフォンです。 |
| TABLET| タブレット端末です。 |
| NONE| 可能なすべてのデバイスに配信します。 |
