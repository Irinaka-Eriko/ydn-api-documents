# DeviceAppType (enum)
DeviceAppTypeは、配信先デバイスの掲出先の種類を表します。
### Service
+ [AdGroupService](../services/AdGroupService.md)

| 値 | 説明 | 
|---|---|
| APP| アプリ向け広告です。 |
| WEB| ウェブ向け広告です。 |
| NONE| ノンターゲティングです。すべてのデバイスに掲出されます。 |
